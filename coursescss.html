Cascading Style Sheet

Introduce CSS

CSS is a styling language (design language), part of a markup language that can "color" or design a web page. CSS itself stands for Cascading Style Sheet. This markup language is usually paired with a markup language such as HTML.

CSS VS
CSS3

CSS uses an old standard colour format. CSS3 provides a different gradient colour and schemes like RGBA, HSLA, HSL, and other colour gradients. CSS does not have the concept of modules. CSS3 incorporates a new feature where it can group CSS.
